### Fix 3 — systemctl is-active comparison bug
- **Files:** `standup-fleet.yml`, `phase5-agent-services.yml`
- **Root:** `systemctl is-active` returns `"active"`, not `"running"`; comparison always false → always showed ❌
- **Fix:** Removed `&& echo "running"` post-processing, compare `pi_running.stdout | trim == 'active'`

- **File:** `deploy-fleet.yml`
- **Root:** `- name:` with task content nested under comment block, not `tasks:` list
- **Fix:** Restructured with proper `tasks:` block
