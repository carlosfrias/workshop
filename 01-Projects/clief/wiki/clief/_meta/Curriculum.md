# Clief Notes Curriculum

The curriculum at [skool.com/cliefnotes](https://www.skool.com/cliefnotes) is structured as progressive levels:

## Level 1: Getting Started

Orientation, community navigation, understanding how leveling works, where to find courses, what to do if you get stuck. Quick — spend a session here and move on.

## Level 2: The Foundation

The core curriculum. Foundational skills for AI use that will remain relevant:
- **Interpretable Context Methodology (ICM):** The central methodology — folder structure as agent architecture
- **5-Layer Context Hierarchy:** CLAUDE.md → CONTEXT.md → Stage CONTEXT → Reference material → Working artifacts
- **Stage Contracts:** Inputs, Process, Outputs — the CONTRACT.md pattern
- **Human Review Gates:** How to build review into every stage boundary
- **Configure the Factory, Not the Product:** Set up once, run repeatedly

## Level 3: The Archive

Three years of evidence validating the methodology. Proves that these frameworks have staying power. Useful for building confidence that ICM isn't a fad — it's built on Unix principles that are 50+ years old.

## Level 4: Implementation Playbooks

Domain-specific build guides. Each module is a complete walkthrough for building an ICM workspace for a particular use case:
- Script-to-animation pipeline
- Course deck production
- Content creation workflows
- Custom domain workspace building

## Level 5: Building Your Stack

Custom interfaces, remote access, infrastructure:
- Building custom UIs on top of ICM workspaces
- Remote access to workspaces
- Infrastructure that wraps around workflows
- Stop adapting to someone else's setup — assemble your own

## Level 6: The Vault

Every asset, template, and production file:
- Workspace blueprints
- CLAUDE.md examples
- Spec templates
- Workflow starters
- Reference docs
- Two structured courses on the 60/30/10 framework

## Level 7: The Drawing Room (VIP)

Live sessions, bespoke folder builds, direct access:
- Live High Tea sessions
- Bespoke folder and workflow builds
- Private Discord channels
- Not a content library — show up, get help, build
