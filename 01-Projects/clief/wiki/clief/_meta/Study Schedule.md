## Phase 4: Ongoing

- The Vault: Templates, blueprints, reference docs (3 sessions/week)
- Drawing Room: Live sessions when available
- Community engagement for leveling

## Session Format

Each 45-minute session:

1. **Review (5 min)** — What did you study last time? What stuck?
2. **Study (25 min)** — Read the assigned material, take notes
3. **Practice (10 min)** — Try one concept hands-on: create a CONTEXT.md, build a stage, etc.
4. **Log (5 min)** — Record what you learned in the activity log

## Progress Tracking

Update the curriculum tracker in the vault domain after every session. Mark completed sections and note insights or questions for next time.
