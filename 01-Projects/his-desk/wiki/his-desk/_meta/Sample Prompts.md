# Sample Prompts

Ready-to-use prompts for each domain. Copy and paste directly — no command-line knowledge needed.

---

## Study Domain

### Passage Analysis
> Study John 15:1-8. Provide observation, interpretation, and application. Compare ESV and NASB translations. Include original language notes on key terms.

### Word Study
> Do a word study on *agape* (Greek ἀγάπη). Show all occurrences in the New Testament, Strong's number, usage patterns, and how it contrasts with *phileo* and *eros*.

### Cross-Reference
> Map all cross-references for Psalm 23. Show how each verse connects to other passages in both Old and New Testaments.

### Commentary Comparison
> Compare commentaries on Romans 8:28. Present at least three interpretive views: Reformed, Arminian, and narrative/historical.

---

## Devotional Domain

### Daily Devotional
> Create a devotional on Psalm 46:10 ("Be still, and know that I am God"). Structure: Scripture → Reflection → Response → Prayer. Use ESV.

### Devotional Series
> Design a 7-day devotional series on the Fruit of the Spirit (Galatians 5:22-23). One fruit per day, each with a focus passage, reflection prompt, and prayer.

### Prayer Guide
> Create a structured prayer guide using the ACTS model (Adoration, Confession, Thanksgiving, Supplication) with scripture references for each section.

---

## Data Domain

### Fetch Bible Text
> Set up a script to download the ESV text of the book of John using the ESV API. Cache the results locally as JSON.

### Build Cross-Reference Database
> Create a SQLite database of cross-references. Include book, chapter, verse, and reference target. Source from an open cross-reference dataset.

### Word Frequency Analysis
> Write a script that counts word frequencies in a given Bible book using a Greek/Hebrew morphological text. Output the top 50 words with Strong's numbers.

---

## Site Domain

### Initial Build
> Initialize an MkDocs site with the Material theme. Set up navigation for study notes, devotional content, word studies, and topical studies. Configure search.

### Deploy
> Build the site locally and verify all links. Then deploy to the configured hosting target.

### Add Navigation
> Add a new section to the site navigation for the book of Romans study series. Create an index page and wire it into the main nav.
