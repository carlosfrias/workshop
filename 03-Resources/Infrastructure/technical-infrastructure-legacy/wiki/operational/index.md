# Technical Infrastructure — Operational

This directory contains **domain-specific operational documentation** for the Technical Infrastructure domain — work product from actual TI sessions, plans, recommendations, and status reports.

⚠️ **Not to be confused with:** `operational/` (workspace root) which contains **cross-domain** operational coordination, PDS templates, and the plan decomposition system.

---

## What's Here

| Subdirectory | Purpose |
|-------------|---------|
| [`planning/`](planning/) | TI-specific plans (PLAN-2026-05-01-* series) |
| [`sessions/`](sessions/) | TI-specific session notes (SESSION-NOTES-2026-05-*) |
| [`status/`](status/) | TI-specific status reports (STATUS-2026-05-*) |
| [`recommendations/`](recommendations/) | TI-specific recommendations |

## What's NOT Here

- **Cross-domain planning templates** → See [`/operational/planning/`](/operational/planning/)

- **Completed backlog archive** → See [`/operational/backlog-completed/`](/operational/backlog-completed/)
