## Known Limitation

`vitepress build` (static site generation) fails with a known VitePress upstream bug when `srcDir` points to symlinked content outside the project root.  
- Symptom: `Rollup failed to resolve import "vue/server-renderer" from "/path/to/wiki/FILENAME.md"`
- Impact: **Static deployment** (GitHub Pages) is currently blocked. **Dev mode** (`vitepress dev`) works perfectly.
- GitHub references: vuejs/vitepress#3919, #3035, #2713
- Workaround for deployment: copy `wiki/` content into `wiki-build/` non-symlinked before build.
