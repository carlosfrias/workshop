## What Is Released

### pi-keyword-router v0.0.1

**Install:**
```bash
pi install github:carlosfrias/pi-keyword-router
```

**Features:**
1. **Auto-complexity routing** — Every prompt classified as trivial/simple/medium/hard before keyword inference
2. **Runtime logging** — Every routing decision appended to JSONL (timestamp, model, cost, confidence)
3. **Auto-decomposition triggers** — HARD prompts generate trigger files for background processing
4. **Dashboard** — `routing_dashboard.py` reads JSONL + legacy JSON, unified reporting
5. **Commands:**
   - `/keyword-route` — router state and history
   - `/keyword-route-log` — last 10 routing decisions
   - `/keyword-route-decompose` — pending/completed decomposition triggers
   - `/keyword-route-off` / `/keyword-route-on` — toggle routing

**Config fields:**
```json
{
  "enableComplexityRouting": true,
  "complexityConfidenceThreshold": 0.55,
  "enableAutoDecomposition": false,
  "decompositionNodePool": ["fnet2", "fnet3", "fnet4", "fnet5", "fnet6"],
  "maxDecompositionSteps": 5
}
```
